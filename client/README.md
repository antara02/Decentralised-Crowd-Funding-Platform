## Quick Started

### 1. Clone/Download the Repository
```
$ git clone https://github.com/abhi1506manu/Decentralized-Crowdfunding-Platform.git
```

### 2. Install Dependencies
```
$ cd client
$ npm install
```

### 3. Start frontend
```
$ npm run dev
```
